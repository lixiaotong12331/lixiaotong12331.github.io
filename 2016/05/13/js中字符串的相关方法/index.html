js中关于字符串的常用方法的总结

js中关于字符串的常用方法的总结

  • substr 从起始索引号提取字符串中指定数目的字符;

  • substring() 提取字符串中介于两个指定下标之间的字符。

  • slice() 方法可提取字符串的某个部分，并以新的字符串返回被提取的部分。

  • charAt() 返回指定位置的字符
    tips: 在javascript中没有字符的概念，所以返回的是长度为1的字符串;

  • replace() 方法用于在字符串中用一些字符替换另一些字符，或替换一个与正则表达式匹配的子串。

  • concat() 方法用于连接两个或多个字符串。

  • indexOf() 方法可返回某个指定的字符串值在字符串中首次出现的位置。

stringObject.split(separator,howmany)
// 参数1：从该参数指定的位置开始分割
// 参数2：分割后返回的子串不会多于这个参数指定的数组

  • toLowerCase() 方法用于把字符串转换为小写。
  • toLocaleLowerCase() 方法用于把字符串转换为小写。
  • toUpperCase() 方法用于把字符串转换为大写。
  • toLocaleUpperCase() 方法用于把字符串转换为大写。
    tips:与 toLowerCase(),,toUpperCase() 不同的是，toLocaleLowerCase() ,toLocaleUpperCase() 方法按照本地方式把字符串转换为小写。只有几种语言（如土耳其语）具有地方特有的大小写映射，所有该方法的返回值通常与 toLowerCase() 一样。
